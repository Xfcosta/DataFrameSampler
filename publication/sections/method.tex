\section{Method}

\subsection{Latent Construction}

Let $D$ be a dataframe with $m$ rows. Columns are split by a simple policy:
numeric columns with more than two non-missing values are continuous; all other
columns are categorical. Thus binary numeric, boolean, and string columns are
categorical targets.

Continuous columns are median-imputed and standardized. Categorical columns are
one-hot encoded for context. For each categorical column $C_j$, DataFrameSampler
removes $C_j$'s own current block from the context and fits
\[
  z_j = f_j(X_{-j}) \in \mathbb{R}^{k_j}
\]
with scikit-learn Neighborhood Components Analysis using the labels of $C_j$ as
the supervised target. The learned block replaces the previous one-hot block.
This process is repeated for a configured number of iterations, yielding
\[
  Z = [X^{num} \mid z_1 \mid \cdots \mid z_q].
\]

\subsection{Decoding}

The inverse map is approximate rather than algebraic. Continuous columns are
inverse-scaled. For each categorical column, a \texttt{RandomForestClassifier}
is trained from its final block $z_j$ to the observed category labels. During
\texttt{inverse\_transform}, \texttt{predict\_proba} provides a distribution
over fitted categories. The decoder either samples from that distribution or
uses the most likely category.

\subsection{Generation}

Generation operates only in the fitted latent matrix. For each synthetic row,
the sampler chooses an anchor $A$, a mutual neighbour $B$, and a mutual
neighbour $C$ of $B$. It then applies
\[
  A' = A + \lambda(C - B).
\]
The resulting latent point is clipped to fitted coordinate ranges and decoded
back to a dataframe row. This is a local translation heuristic. It assumes the
supervised latent space is locally linear enough for displacement transfer to
remain plausible, but the method does not guarantee manifold preservation or
constraint satisfaction.

\section{Introduction}

Tabular datasets remain a central representation in scientific computing,
business analytics, health informatics, and machine learning. In many practical
settings, developers and analysts need data that resembles an existing table in
order to test software, benchmark downstream workflows, or demonstrate tools.
However, direct reuse of the original data may be undesirable because of
privacy, licensing, operational, or convenience constraints.

DataFrameSampler is a small Python package designed for this pragmatic setting.
Its goal is not to provide formal privacy guarantees, but to offer a simple
mechanism for generating dataframe-shaped samples that preserve some of the
local structure of the input table. The package accepts pandas dataframes, CSV
files, or Parquet files, and produces generated samples with the same column
interface.

The central idea is to represent each row in a discretized latent bin-space.
Numeric columns are discretized directly. Non-numeric columns are first mapped
to numeric values, either by frequency encoding or by embedding selected numeric
helper columns into one dimension. Synthetic latent rows are generated by
sampling local neighbour chains and applying a difference vector observed among
neighbours. The resulting latent row is then decoded by sampling original
observed values from corresponding bins.

The current software also includes engineering features intended to make the
method usable in modern data workflows: selectable embedding algorithms from
scikit-learn, exact and approximate nearest-neighbour backends, optional OpenAI
assisted configuration, and extension-based CSV/Parquet serialization.

The contributions of this work-in-progress paper are:

\begin{itemize}
  \item a concise description of latent bin-space sampling for mixed-type
  dataframe generation;
  \item an open-source Python implementation with a command-line interface and
  library API;
  \item a configurable categorical vectorization mechanism based on helper
  columns and one-dimensional embeddings;
  \item a planned evaluation protocol for comparing generated and source
  tables across distributional and predictive tasks.
\end{itemize}

\section{Synthesis Layer}

The strongest conclusion supported by the evidence is a practical-use claim.
DataFrameSampler is a small, mixed-type dataframe sampler whose behaviour can
be inspected through fitted latent blocks, analogy traces, decoder
probabilities, rejection counts, and ordinary dataframe outputs. Each generated
row can be explained constructively as an anchor modified by an observed
neighbour-pair displacement, which is a stronger interface than merely reporting
that a sample came from a fitted black-box distribution. The evaluation
therefore supports the claim that the method can be useful for tests,
demonstrations, notebooks, and exploratory workflows when low setup cost and
auditable construction are more important than formal release guarantees.

The mechanism claim remains narrower. Per-column NCA blocks provide mechanism
evidence only when they improve held-out categorical prediction against
majority and same-width PCA references, and the concatenated latent matrix has
no joint objective that would guarantee compatibility across all blocks.
Iterative refinement can improve the context available to later projections,
however it can also propagate earlier approximation errors, and there is no
monotonic criterion that certifies improvement across iterations. In other
words, the NCA construction is a structured empirical representation, not a
proof that the learned geometry is globally optimal.

The decoder and generator define the main boundary conditions. Random forests
approximate $p(C_j \mid Z_{-j})$ independently for each categorical column,
where $Z_{-j}$ denotes the final latent context with the target block removed;
optional calibration and calibration diagnostics expose probability quality but
do not impose cross-column consistency. It can therefore occur that a generated
row is locally plausible for each decoder while violating a structured
constraint that spans several columns. Likewise, a mutual-neighbour displacement
is easy to inspect because it can be described as $D:A\approx C:B$, but that
same simplicity means the analogy is an observed latent displacement, not a
semantic or causal explanation and not a formal manifold sampler.

The allowable conclusion is therefore deliberately conservative. The paper
does not establish general superiority over synthetic-data systems, formal
privacy protection, or guaranteed manifold preservation. It establishes a
bounded empirical pattern: under the studied workflows and diagnostics,
DataFrameSampler provides an inspectable example-data generator whose useful
behaviour can be measured, whose mechanism can be partially ablated, and whose
failure modes remain visible enough to guide cautious use.

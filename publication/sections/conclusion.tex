\section{Conclusion}

DataFrameSampler 2.0 is a lightweight mixed-type dataframe sampler based on
iterative supervised NCA latent blocks and local mutual-neighbour transport. It
provides a direct sklearn-style API and a CLI while keeping the modelling claim
narrow: the method is simple, inspectable, and useful for example-data
workflows, but it is heuristic and not privacy-preserving.

Future work should evaluate stability of the iterative latent construction,
constraint-aware decoding, stronger diagnostics for high-cardinality
categoricals, and alternatives to independent per-column decoders.

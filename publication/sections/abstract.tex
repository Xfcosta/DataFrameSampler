Synthetic tabular data generation is often useful for testing pipelines,
demonstrating software, and exploring modelling workflows when direct use of
source data is inconvenient. This paper presents DataFrameSampler, a lightweight
Python library for generating mixed-type tabular samples from pandas dataframes.
The method maps numeric and categorical columns into a discretized latent
bin-space, samples new latent rows by transferring differences along local
neighbour chains, and decodes latent bins back into observed column values. The
implementation supports configurable categorical vectorization, exact and
approximate nearest-neighbour backends, CSV and Parquet data I/O, and optional
LLM-assisted configuration. This manuscript is a work-in-progress stub and
outlines the method, software interface, and planned empirical evaluation.

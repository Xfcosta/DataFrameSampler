Synthetic tabular data generation is useful when teams need realistic examples
for software tests, demonstrations, notebooks, and exploratory modelling but
cannot casually reuse source data. This paper presents DataFrameSampler 2.0, a
lightweight Python library for mixed-type pandas dataframes. The method learns a
fully numerical latent space by fitting one supervised Neighborhood Components
Analysis block for each categorical column. Binary columns are categorical even
when stored as numeric values. Generation applies a local mutual-neighbour
translation in the learned latent space and decodes the result with inverse
numeric scaling plus probabilistic random-forest categorical decoders. The
method is simple and auditable, but heuristic: it has no global objective, no
formal manifold-preservation guarantee, and no privacy guarantee.

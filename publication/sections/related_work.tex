\section{Related Work}

Synthetic tabular data generation includes probabilistic graphical models,
copulas, nearest-neighbour resampling, oversampling methods such as SMOTE, and
deep generative models such as CTGAN \cite{chawla2002smote,xu2019ctgan}. Larger
systems such as SDV provide practical interfaces for several model families
\cite{patki2016sdv}.

DataFrameSampler sits closer to local interpolation and metric-learning
methods. It uses scikit-learn components, including Neighborhood Components
Analysis and nearest-neighbour search \cite{pedregosa2011scikit}. Optional ANN
backends such as NN-Descent, HNSW, and Annoy can be used for larger datasets
\cite{dong2011efficient,malkov2018hnsw,bernhardsson2015annoy}.

Privacy-preserving synthetic data addresses a stronger problem than this
package claims to solve. Formal release mechanisms require explicit threat
models, privacy definitions, and disclosure-risk evaluation.

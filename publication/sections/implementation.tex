\section{Implementation}

The public library entry point is \texttt{DataFrameSampler}. The estimator
accepts pandas dataframes in \texttt{fit} and \texttt{transform};
\texttt{transform} returns a NumPy latent matrix; \texttt{inverse\_transform}
accepts a NumPy matrix and returns a dataframe; \texttt{generate} returns a
synthetic dataframe.

The implementation is organized around:
\begin{itemize}
  \item \texttt{sampler.py}: NCA latent learning, decoding, model persistence,
  and neighbour-transport generation;
  \item \texttt{neighbours.py}: mutual-neighbour estimation and sampling
  probabilities;
  \item \texttt{knn.py}: exact, scikit-learn, and optional approximate nearest
  neighbour backends;
  \item \texttt{io.py}: CSV and Parquet dataframe I/O;
  \item \texttt{cli.py}: file-oriented command-line workflow.
\end{itemize}

Version 2.0 exposes one direct estimator and removes auxiliary configuration,
column-selection, embedding-method-selection, and bin-encoding workflows. The
CLI therefore exposes only the NCA and neighbour-transport parameters.
